# Zeus
Simple Task Scheduler with NodeJs 

- Build Tools
Runs on Node
Scheduling using node-cron
Uses Webpack 4


## Installation

```
git clone https://github.com/saikota/NodeJS-Webpack-Babel.git
npm install
```

## Config

```
/src/config
```

## Build development

```
npm run dev:build
```

## Build production

```
npm run prod:build
```

## Run on development

```
npm run zeus
```
